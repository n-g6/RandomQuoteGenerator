What is this app?

This little app is the first of 20 programming projects I have set out to do.

It operates by fetching a quote from an API found on the internet, transforming it a tiny bit and displaying it in a readable and pleasant to look at format.

A bit of practice with the "onload" and "onclick" events can be seen through the two types of quotes which generate:

  • The top ones which generate on page load
  • The bottom ones which generate at the click of the "Generate" button

Due to my lack of almost any web-focused JS knowledge, the script is pretty basic and is in part written while following tutorials.
